# 第一周 Python AI工程基础

## 课程主题
**从Python开发者成长为AI工程师——理解AI开发的完整流程**

## 本周目标
通过一个完整的线性回归案例，理解一个AI项目从数据处理、特征工程、模型训练到模型评估的完整开发流程，建立AI工程开发的基本认知。

本周结束后，希望学生能够回答下面几个问题：
* AI项目一般包含哪些步骤？
* 什么是Feature（特征）？
* 为什么需要特征工程？
* 什么是训练集和测试集？
* 一个模型是如何训练出来的？
* 如何评价模型效果？
* 如何利用模型进行预测？

## 学习内容
（关于概念和名词，最快的学习方法是通过和大模型问答的方式来理解，并且通过动手实践来增加理解）

### 1. AI开发环境
* **核心工具与平台：**
  * Python开发环境配置
  * Conda环境管理（环境隔离与依赖控制）
  * VS Code编辑器高效使用
  * Jupyter Notebook交互式开发
* **环境配置步骤：**
  * 参考同文件夹下的文件：开发环境Setup.md

### 2. AI开发流程
理解一个完整AI项目的基本Workflow（工作流）：

```

Data (数据)
↓
Feature Engineering (特征工程)
↓
Model Training (模型训练)
↓
Evaluation (模型评估)
↓
Inference (推理预测)

```

### 3. Python数据分析
* **核心库学习：** NumPy、Pandas、DataFrame、Matplotlib
* **关键概念与核心能力：**
  * **EDA (Exploratory Data Analysis)：** 探索性数据分析
  * **数据可视化：** 绘制分布图、相关性图表等
  * **数据统计分析：** 均值、中位数、标准差等基础统计量

### 4. 特征工程
* **概念理解：** Feature（特征）与 Label（标签）的关系
* **常用数据处理技术：**
  * 数据清洗与缺失值处理
  * **Feature Encoding：** 特征编码（如独热编码、标签编码）
  * **Normalization：** 规范化/归一化（如缩放特征区间）
  * **Train/Test Split：** 训练集与测试集划分（防止模型过拟合）

### 5. 第一个机器学习模型
* **框架与算法：** scikit-learn 库、LinearRegression（线性回归模型）
* **核心步骤：** Model Training（模型训练）、Prediction（推理预测）
* **模型评估指标：**
  * **MSE** (Mean Squared Error) 平均平方误差
  * **RMSE** (Root Mean Squared Error) 均方根误差
  * **MAE** (Mean Absolute Error) 平均绝对误差
  * **R² Score** (Coefficient of Determination) 决定系数

## Project Workflow (项目推进五步法)

### Step 1：Run（运行项目）
* **目标：** 成功运行整个项目，建立开发环境。
* **完成内容：**
  1. 下载GitHub项目到本地
  2. 配置Python环境并安装相关依赖
  3. 成功运行参考的Notebook
  4. 浏览整个项目目录
* **思考题：** 一个AI项目通常由哪些文件组成？
* **输出成果：** 项目在本地能够正常顺畅运行。

### Step 2：Understand（理解代码与数据）
* **目标：** 理解整个AI项目在做什么。
* **完成内容：** 深度阅读并剖析Notebook，重点理解：
  * 数据来源与数据结构
  * Feature与Label的定义
  * 数据探索（EDA）的具体实现
  * 特征工程的策略与方法
  * 模型训练与评估的完整流程
* **输出成果：** 完成一份详实的项目阅读笔记。

### Step 3：Modify（修改与扩展）
* **目标：** 在原项目基础上进行修改，真正吃透代码逻辑。
* **完成内容：** 尝试进行以下修改并观察对模型最终结果的影响：
  * 增加新的EDA可视化图表
  * 修改原有的数据清洗或缺失值处理方式
  * 构建或增加新的Feature
  * 修改Train/Test的划分比例
  * 调整模型的超参数
* **输出成果：** 记录并对比新的实验结果。

### Step 4：Build（独立实现核心功能）
* **目标：** 脱离原Notebook，自己从零重新实现核心AI流程。
* **完成内容：** 独立构建如下流程：

```

读取数据 → EDA → Feature Engineering → Train/Test Split → Linear Regression → Evaluation → Prediction

```
* **规范项目目录：**
```text
week01_house_price/
├── data/           # 原始数据与处理后的数据
├── notebooks/      # 实验与探索性的Notebook
├── src/            # 核心Python源代码
├── models/         # 训练好的模型持久化文件
├── figures/        # 导出的EDA及评估图表
├── README.md       # 项目说明文档
└── requirements.txt# 依赖包列表

```

* **输出成果：** 拥有完全独立自主实现的 House Price Prediction 工程项目。

### Step 5：Summarize（总结与工程化）

* **目标：** 完成一个规范的、可交付的AI工程项目。
* **完成内容：**
* 整理整个项目结构，去除冗余文件
* 编写结构完整的 README 说明文档
* 导出并整理 `requirements.txt`

* **README 框架建议：**
